#### Parameters

| Name | Type | Description |
| :------ | :------ | :------ |
| `actions` | [`Action`](../interfaces/Action.md)[] | An array of `Action` objects from which to extract conditions. |

#### Returns

`string`

A string listing each action's name and its condition.
